微信开发循环原则：
    基于campus-card-wechat-api,campus-card-wechat-web目录来开发，UI页面布局已手机端做适配，已图标分类展示各个应用页面，页面框架采用WeUI，底部导航：首页，消费，充值，我，四大块

微信手机端开发规范文档
一、整体架构与导航设计
1. 四大核心导航模块
首页：聚合核心信息与快捷入口，如子女在校概览、校园动态等。
消费：展示消费记录、消费分类统计等功能。
充值：提供校园卡充值、充值记录查询等服务。
我：用户信息管理、设置、帮助中心等功能集合。
2. 底部导航栏规范
视觉风格：采用微信原生风格的线性图标 + 文字组合，选中态为蓝色（#2778F1），未选中态为灰色（#999999）。
尺寸要求：图标尺寸24px×24px，文字字号12px，导航栏整体高度50px，确保各机型点击区域充足。
交互规则：点击导航项时，需有 “微缩 + 颜色切换” 的反馈动画，增强用户操作感知。
二、各页面详细开发规范
1. 首页页面规范
（1）布局结构
顶部栏：左侧返回箭头（微信风格 “<”），中间标题 “家长端首页”，右侧消息通知图标（铃铛样式）。
用户信息区：家长头像（圆形，直径60px）、姓名（ID：XXX）横向排列，下方 “绑定子女” 板块（单选按钮选择子女，选中态蓝色#2778F1）。
核心功能区：“校园一卡通” 卡片式模块，包含 “消费记录”“充值入口” 快捷按钮（搭配线性图标，如钱包、加号 + 钱包）。
底部导航栏：固定 “首页”“消费”“充值”“我” 四个选项。
（2）视觉与交互
字体：标题18px（字重 Medium），信息说明14-16px（字重 Regular）。
颜色：背景#FFFFFF，模块分隔线#EEEEEE，确保页面清爽整洁。
交互：卡片式功能区点击时有轻微 hover 效果（背景色#F5F7FA）。
2. 消费页面规范
（1）布局结构
顶部栏：左侧返回箭头，中间标题 “消费记录”，右侧可添加 “筛选” 图标（漏斗样式）。
数据展示区：消费列表（按时间倒序排列），每条记录包含消费时间、消费场景（如食堂、超市）、消费金额、消费状态（成功 / 失败）。
统计分析区：消费分类饼图 / 柱状图（采用轻量图表库，如wx-charts），展示月度消费分布。
底部导航栏：同全局导航规范。
（2）视觉与交互
列表样式：每条消费记录采用 “左图标（场景标识）+ 中文字信息 + 右金额” 布局，交替行背景色#F9FAFB增强可读性。
图表交互：点击图表区域可弹出详情浮层，展示更细分的消费数据。
加载规则：采用微信原生下拉刷新、上拉加载更多的交互逻辑。
3. 充值页面规范
（1）布局结构
顶部栏：左侧返回箭头，中间标题 “校园卡充值”，右侧 “充值记录” 入口。
充值金额区：预设金额按钮（如 50 元、100 元、200 元）+ 自定义金额输入框，支持微信支付唤起。
充值记录区：按时间倒序展示充值金额、支付方式、充值状态。
底部导航栏：同全局导航规范。
（2）视觉与交互
金额按钮：圆角矩形（圆角8px），选中态背景#2778F1、文字白色，未选中态背景#F5F7FA、文字#333333。
支付交互：点击充值按钮后，弹出微信支付授权框，支付成功后返回充值结果页并给出明确提示（如 “充值成功，余额 XXX 元”）。
4. “我” 页面规范
（1）布局结构
顶部栏：左侧返回箭头，中间标题 “个人中心”。
个人信息区：家长头像（圆形，直径80px）、姓名、ID 号，下方 “绑定子女管理” 入口。
功能列表：设置、帮助中心、关于我们等选项，采用微信原生列表样式（左图标 + 右箭头）。
底部导航栏：同全局导航规范。
（2）视觉与交互
信息编辑：点击头像可进入头像更换页面，点击姓名 / ID 可进入信息编辑页（支持修改绑定手机等信息）。
列表交互：每个功能项点击时有 “点击反馈”（背景色短暂变灰），增强交互感。
三、通用开发规范
1. 适配规范
采用微信小程序rpx单位或 H5rem单位，确保在不同机型（如 iPhone、安卓）上布局自适应。
图片资源采用 WebP 格式（若兼容）或压缩后的 JPG/PNG，保证加载性能。
2. 性能规范
页面首屏加载时间控制在 2 秒内，列表页采用 “懒加载” 或 “分段加载”。
避免不必要的 DOM 节点嵌套，微信小程序需注意逻辑层与视图层的通信效率。
3. 交互规范
所有可点击元素需有明确的 “点击态”（颜色变化、微动效）。
弹窗、浮层需有 “出现 / 消失” 动画，避免突兀切换。
4. 视觉规范
色彩体系：主色#2778F1（用于按钮、选中态），辅助色#666666（用于说明文字）、#EEEEEE（用于分隔线），中性色#333333（用于标题）。
字体体系：标题18px（字重 Medium），正文16px（字重 Regular），辅助文字14px（字重 Light）。
